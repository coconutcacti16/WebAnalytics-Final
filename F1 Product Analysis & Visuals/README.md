# F1 Product Analysis & Visuals

This folder contains visual analyses of F1 product data, focusing on price dynamics, top team products, and demographic distributions.

## Contents
- **Visuals:** `combined_historical_price_dynamics.png`, `gender_distribution_trends.png`, `top_products_per_team_*.png`
- **Scripts:** `analyze_all_years.py`, `analyze_prices.py`, `visualize_top_queries.py`

## How to Run
To generate the historical price dynamics visualizations across all years:
```bash
python analyze_all_years.py
```
To analyze price fluctuations and generate impact visualizations:
```bash
python analyze_prices.py
```
