# F1 Google Trends & Analysis

This folder contains data and visualizations related to F1 Google Trends search queries from 2021 to 2025. It includes monthly interest metrics and top related queries.

## Contents
- **CSV Data:** `f1_monthly_interest_2021_2025.csv`, `f1_race_weekend_trends_2021_2025.csv`, `f1_top_queries_2021_2025.csv`, `race_schedule_2021_2025.csv`
- **Visuals:** `f1_interest_trends_visual.png`, `f1_top_queries_visual_improved.png`
- **Scripts:** `visualize_top_queries.py`

## How to Run
To generate the improved top queries visualization, run:
```bash
python visualize_top_queries.py
```
